Viewing other user accountsÂ¶

Now that your account is created, you can view other accounts on the system. Note on the main profile page there are options for following (and blocking) other users.

  1. To see information about other users on the system, click the People link on the top toolbar.

  2. You will see a list of users registered on this system. Click on the user name for a particular user.

    List of users

  3. You will see the layers owned by this user. You can also click Activities to see the activity feed.

    List of layers owned by a user

    List of users

  4. If you are interested in keeping track of what this user does, click the Follow button. A confirmation page will display. Click Confirm.

    Confirming following a user

  5. You will now be following this user, and your profile page will note this.

    Success following a user
