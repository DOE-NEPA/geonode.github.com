GeoNode Users Workshop¶

Welcome to the GeoNode Users Workshop! This workshop will teach how to use the GeoNode software application.

These workshop materials should be accompanied by a data package containing the following files:

  • /doc—These workshop materials in HTML format
  • /data—Spatial data to be used during the workshop
