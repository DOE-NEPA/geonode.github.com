Open Source Geospatial

Open from top to bottom

GeoNode is built upon a platform of proven open source components including Django, GeoServer, pycsw, OpenLayers and GeoExt. GeoNode implements many Open Geospatial Consortium (OGC) standards, including WMS, WFS, WCS, KML, and CSW. GeoNode development also contributes to the underlying open source projects and software libraries.
Components include:

  • PostGIS Spatial Databases
  • GeoServer OGC Services
  • pycsw CSW Metadata Catalogue
  • Geospatial Python libraries
  • OpenLayers and GeoExt Web Mapping Libraries
Learn about GeoNode's Open Source Geospatial Foundation

Django

Built for the web

    GeoNode leverages the Django web Framework, offering a friendly and extensible environment for web developers accustomed to using any modern MVC web framework. In contrast with many geospatial tools which force web developers to adapt to a "GIS" specific stack, GeoNode and Django offer a framework on which developers can build Geospatial web applications using tools they are already comfortable and familiar with.

    Django brings its large user base and ecosystem of apps, making it easy to develop modern, social websites.

    Features Include:

  • Modern, high productivity web framework
  • Pluggable GeoNode Django apps
  • Deployable Template GeoNode projects
  • Integrated Admin Interface
  • Large ecosystem of third-party pluggable apps
  • Spatial GeoDjango ORM Available
Learn about GeoNode's pluggable Django apps and deployable projects

Bootstrap and jQuery

Modern and flexible web front-end

GeoNode's front end is based on the popular and flexible jQuery and Bootstrap frameworks. A set of jQuery javascript plugins is used to drive the interactive and responsive user interface, and Bootstrap css is used to style the pages. Both of these frameworks make it easy theme and customize your own GeoNode implementation and since both are widely used in other projects, integration is frictionless.


Features include:

  • Bootstrap CSS Framework
  • jQuery javascript front end and plugins
  • Django templating system
  • Themable with lessc and bootswatch
Learn about GeoNode's Front End

APIs

Integrate easily

Because of it's solid foundation in open source GeoNode supports multiple points for application integration.
Features include:

  • OGC Compliant Web Services
  • GeoServer REST API
  • GeoNode Search and REST APIs
  • Tile Map Service (TMS)
Learn about GeoNode's APIs

Interoperability

GeoNode plays well with others

GeoNode is not only a geospatial platform, it's a social one. GeoNode's foundational components make it easy to interoperate with other GeoNodes and OGC Services as well as with many other popular social services and tools.

  • Federates with other GeoNode's
  • Federates with other OGC Compliant SDI (CSW/WMS)
  • Federates with other Open Data Platforms (ckan, junar, socrata)
  • Integrates with other Social Services & Platforms (Twitter, Facebook, Google+, LinkedIn, Jive, Telligent)
  • Integrates with other CMS (Wordpress, Drupal)
