Open Source Geospatial

Solid foundation on Open Source Geospatial

  • Based on reliable, mature open source geospatial software
  • PostGIS Spatial Databases
  • GeoServer OGC Services
  • pycsw CSW Metadata Catalogue
  • Geospatial Python libraries
  • OpenLayers and GeoExt Web Mapping Libraries
Learn about GeoNode's Open Source Geospatial Foundation

Django

Built on the Django web Framework

  • Modern, high productivity web framework
  • Pluggable GeoNode Django apps
  • Deployable Template GeoNode projects
  • Integrated Admin Interface
  • Large ecosystem of third-party pluggable apps
  • Spatial GeoDjango ORM Available
Learn about GeoNode's pluggable Django apps and deployable projects

Bootstrap and jQuery

Modern and flexible web front end

  • Bootstrap CSS Framework
  • jQuery javascript front end and plugins
  • Django templating system
  • Themable with lessc and bootswatch
Learn about GeoNode's Front End

APIs

Multiple points for application integration

  • OGC Compliant Web Services
  • GeoServer REST API
  • GeoNode Search and REST APIs
  • Tile Map Service (TMS)
Learn about GeoNode's APIs

Interoperability

Integration with external Applications & Platforms

  • Federates with other GeoNode's
  • Federates with other OGC Compliant SDI (CSW/WMS)
  • Federates with other Open Data Platforms (ckan, junar, socrata)
  • Integrates with other Social Services & Platforms (Twitter, Facebook, Google+, LinkedIn, Jive, Telligent)
  • Integrates with other CMS (Wordpress, Drupal)
