Spatial Data Discovery

Browse and search for geospatial data

By enabling the kind of collaboration that has previously been limited to social networks and combining it with specialized geospatial tools, GeoNode makes it easy to explore, discover, process, style, combine and share geospatial data and maps.

By providing a non-technical user focused experience, spatial datasets can be easily imported and shared publicly or restricted to allow access to only specific users or groups.

Features include:

  • Powerful Spatial Search Engine
  • Federated OGC Servers
  • Metadata Catalogue
Learn about GeoNode's Discovery Features

Import & Manage

Upload and securely share geospatial data

GeoNode makes it easy to upload and manage spatial data layers on the web. Any user can upload data layers and make them available via OGC services. Data Layers are available for browsing, searching, styling, processing and combining to make maps that can be shared with specific users or publicly.

Supported upload formats include Shapefiles, GeoTiffs, KML and CSV. Your GeoNode administrator can also help you connect to existing spatial databases, services and specialized raster and vector formats
Features include:

  • Import and publish raster, vector and tabular data
  • Manage layers Metadata and associated Documents
  • Securely or Publicly Share data
  • GIS Data Editor (versioning)
Learn about GeoNode's Data Management Features

Interactive Mapping

Create and Share Interactive Web Maps

GeoNode comes with helpful cartography tools for styling and composing maps through the web. These tools make it easy for anyone to assemble a browser based mapping application with functionality traditionally found in the desktop GIS world.

Users can add and style, style and publish layers with ease. By leveraging GeoExplorer users gain enhanced interactivity with GIS specific tools like identify, query, measure, create feature, and more.
Features include:

  • GeoExplorer GIS Client
  • Interactively Style Editor
  • Create multi-layer and service maps
  • Share and embed interactive maps
Learn about GeoNode's Mapping Features

Collaboration

Social & Collaborative features

GeoNode is all about collaboration. With feature-rich web-based editing and map styling, easy data sharing, and OGC-compliant service endpoints, GeoNode enables the kind of collaboration that has previously been limited to social networks and specialized geospatial tools.

GeoNode is designed to share data simply and easily through crating collaborating.
Features include:

  • Review, rate, and comment
  • User groups
  • Activity stream
  • Announcements and notifications
